🌿 Peace Hub

Peace Hub is a simple mental wellness web application designed to help users relax, manage stress, and express their thoughts in a calm and supportive environment.

The platform provides interactive tools that encourage mindfulness, emotional reflection, and healthy digital habits.

✨ Features
🌬️ Breathing Exercise – Guided breathing animation to help reduce stress
🫙 Worry Jar – Write and release your worries
📓 Journal Vault – Save personal thoughts using browser local storage
🫧 Bubble Pop Game – A relaxing mini-game for stress relief
⏳ Digital Detox Timer – Helps users take mindful breaks from screens
📊 Wellness Insights – View simple activity statistics from your usage
🛠️ Tech Stack
HTML
CSS
JavaScript
Local Storage
🎯 Purpose

This project was created to practice frontend web development and build a useful tool that promotes mental wellness, relaxation, and mindful habits through simple interactive features.
